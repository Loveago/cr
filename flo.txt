Build a full-stack, production-ready cryptocurrency exchange platform similar to Coinbase and Binance with a strong focus on performance, security, scalability, and a modern UI/UX.

Tech Stack:
- Frontend: Next.js (latest, App Router, TypeScript, Tailwind CSS, ShadCN UI)
- Backend: Node.js with Express (TypeScript)
- Database: PostgreSQL (use Prisma ORM)
- Realtime: WebSockets (for live price updates and trades)
- Caching: Redis
- Storage: AWS S3 (for documents, KYC uploads)
- Authentication: JWT + Refresh Tokens + 2FA (Google Authenticator)
- Payments: Stripe + Crypto wallets integration
- Blockchain Integration: Use APIs (like Binance, Coinbase, or Web3 providers)
